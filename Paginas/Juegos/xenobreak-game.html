Game description

In XenoBreak you will have to infiltrate an alien base and stealthily retrieve the key to the power they are hiding. Beware of the cameras and the enemies! Try not to get caught! However, they will inmediatily chase after you once you get ahold of the key, so you better run faster than them!


Other information

This game was developed using Unreal Engine as a university project.


Minimum requirements

  • Windows 10
  • GTX 970
  • Intel core i7
  • 8GB RAM
